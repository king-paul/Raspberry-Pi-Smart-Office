Forms Module¶

This module contains classes used for Flask WTForms.

class forms.PatientForm(formdata=<object object>, **kwargs)¶

This is a Flask WTF form used to create a patient registraion from and validate it. It is rendered on the patients.html page via the ‘/patients’ route.

To use this class make a call to PatientForm()

class forms.AppointmentForm(formdata=<object object>, **kwargs)¶

This is a Flask WTF form used to create a appointment scheduling from and validate it. It is rendered on the patient.html page via the ‘/patients/<id>’ route.

To use this class make a call to AppointmentForm()

forms.get_doctors()¶

This function is used to make a call to the crud API to return all doctors. I then returns a dictionary containing the id and name of each one. This is intended for use with a dropdown combo box on a web form/
